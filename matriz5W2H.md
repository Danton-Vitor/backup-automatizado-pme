# 📋 Matriz 5W2H - Backup Automatizado para PMEs

A matriz 5W2H é uma ferramenta de planejamento estratégico que mapeia as principais diretrizes de execução do projeto por meio de sete perguntas fundamentais.

---

### 📊 Plano de Ação 5W2H

| Sigla | Conceito | Detalhamento do Projeto |
| :---: | :--- | :--- |
| **What** | **O que será feito?** | Implementação de uma rotina de backup automatizada, criptografada em nuvem e isolada da rede local. |
| **Why** | **Por que será feito?** | Para mitigar o risco de perda definitiva de dados corporativos provocada por ataques de *Ransomware* ou falhas físicas de hardware. |
| **Where** | **Onde será feito?** | No servidor local/máquina virtual da empresa cliente com destino automatizado para buckets na nuvem (AWS S3 ou Google Cloud). |
| **When** | **Quando será feito?** | O desenvolvimento e a modelagem iniciam-se em 2026, estendendo-se com os testes e homologação até a apresentação final para a banca do TG em **junho de 2027**. |
| **Who** | **Quem fará?** | Projeto executado de forma **individual** pelo aluno Danton Barbosa do curso de ADS (FATEC Franca). |
| **How** | **Como será feito?** | Utilizando ferramentas *Open Source* para sincronização, aplicando chaves de criptografia simétrica AES-256 e agendamento de tarefas do SO. |
| **How Much** | **Quanto custará?** | Custo de desenvolvimento zero (ferramentas gratuitas) e custo de nuvem variável baseado no consumo real de armazenamento (estimado entre $5 e $20 mensais). |
